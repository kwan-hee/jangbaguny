# 쇼핑몰 장바구니 만들기

#📌배포주소 :
# 프로젝트 설명
* main 화면에서 물건을 선택한 다음 장바구니에 담습니다.
* 장바구니에 담은 물건을 선택한 물건만을 계산합니니다.
* 최신순, 가격순으로 정렬 합니다. 

# 언어
* JavaScript
* React.js

# 라이브러리
* styled-components
* react-dom
* react-router-dom
* prettier
* media quary
